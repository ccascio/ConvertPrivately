# React example

Vite + React showing all 10 pilot converters from `@convertprivately/react`.

```bash
npm install
npm run dev
```

Open http://localhost:5173.

## SEO

`index.html` carries `<meta name="robots" content="noindex, nofollow">` so deploying this example does not compete with [convertprivately.com](https://convertprivately.com) in search results.
